闭包

当内部函数在定义它的作用域外部被引用时，就会创建该内部函数的闭包。如果内部函数引用了位于外部函数的变量，当外部函数调用完毕后，这些闭包中的变量不会被释放，会保持在内存中。

闭包对应函数示例，当函数示例销毁时才会销毁闭包，并释放其占用的内存。
